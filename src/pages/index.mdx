# Certify The Web 
## Documentation & Guidance

**Certify The Web** is our range of automated SSL/TLS certificate management tools for Windows, Linux, macOS, and cloud environments. Choose the solution that fits your needs:

<Tabs queryString="product">
  <TabItem value="ccm" label="Certificate Manager" default>

    Comprehensive certificate management on *Windows Server* with a desktop UI and background management service.

    **Key Features:**
    - Automated ACME certificate renewals
    - Pre-built deployment tasks
    - Support for Let's Encrypt, Google Trust Services, and enterprise CAs
    - Custom automation scripts

    [**Get Started →**](docs/intro)

    ![Certify Certificate Manager](/assets/screens/Startup_Dark_sm.png)

  </TabItem>
  
  <TabItem value="hub" label="Management Hub">
    
    Centralized cross-platform web-based certificate management.

    **Key Features:**
    - Web-based UI accessible from anywhere
    - Certificate management with the same core features as Certify Certificate Manager.    
    - Managed DNS challenges
    - Support for Windows, Linux, macOS, and containers
    - Optionally administer many instances of Certify Certificate Manager and Certify Management Agent.

    [**Get Started →**](docs/hub/)

    ![Certify Management Hub](/assets/screens/hub/hub_intro.png)
    
  </TabItem>
  
  <TabItem value="dashboard" label="Dashboard">
    
    Hosted dashboard for centralized certificate monitoring across your organization.

    **Key Features:**
    - Real-time certificate status monitoring
    - Compatible with multiple ACME clients via Certify Management Agent
    - Organization-wide visibility

    [**View Documentation →**](docs/dashboard)

    ![Dashboard](/assets/screens/dashboard/dashboard_example_sm.png)
        
  </TabItem>
  
  <TabItem value="dns" label="Certify DNS">
   
    Cloud-based DNS challenge service for environments where direct DNS automation isn't possible.

    **Key Features:**
    - CNAME delegation support
    - Compatible with most ACME clients
    - Cross-platform compatibility
    - Simplified DNS challenge management

    [**Setup Guide →**](docs/dns/providers/certifydns)

  </TabItem>
</Tabs>
